# Learning Path for Aisha

**Goal:** Become a data analyst within 6 months

**Skill gap analysis:** Aisha lacks programming skills, data analysis knowledge, and experience with relevant tools and technologies, which are essential for a data analyst role. She has no prior coding experience, making it crucial to start with foundational courses. Her background in commerce suggests she may have some familiarity with data, but this needs to be developed into analytical skills.

## Recommended Path

1. **Python Fundamentals** (`C101`)
   - Reason: Aisha needs to learn programming basics, and Python is a fundamental language for data analysis, making this course a necessary starting point.
2. **Data Analysis with Pandas** (`C202`)
   - Reason: After learning Python, Aisha should focus on data analysis skills, which are directly relevant to her goal of becoming a data analyst. Pandas is a key library for data manipulation and analysis in Python.
3. **Statistics & Probability for Data Science** (`C201`)
   - Reason: Understanding statistics and probability is crucial for a data analyst. This course will provide Aisha with the necessary knowledge to work with data effectively and make informed decisions.

**Estimated total duration:** 13 weeks
