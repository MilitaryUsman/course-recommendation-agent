# Learning Path for Priya

**Goal:** Become a full-stack web developer

**Skill gap analysis:** Priya lacks knowledge in backend development, databases, and full-stack development, which are crucial for becoming a full-stack web developer. She also needs to learn JavaScript, which is a fundamental skill for web development. Additionally, she requires knowledge of React and Node.js to build robust frontend and backend applications.

## Recommended Path

1. **Web Development with JavaScript** (`C204`)
   - Reason: Priya needs to learn JavaScript to build web applications, and this course provides a comprehensive introduction to web development with JavaScript.
2. **React & Frontend Engineering** (`C205`)
   - Reason: To become a full-stack web developer, Priya needs to learn React, a popular frontend framework, and this course teaches React and frontend engineering.
3. **Backend APIs with Node.js** (`C206`)
   - Reason: Priya needs to learn Node.js to build robust backend applications, and this course teaches backend APIs with Node.js, which is essential for full-stack development.
4. **Full-Stack Capstone Project** (`C304`)
   - Reason: This capstone project course allows Priya to apply her knowledge of React, Node.js, and full-stack development to build a real-world project, solidifying her skills and preparing her for a career as a full-stack web developer.

**Estimated total duration:** 26 weeks
